# Decision Log

## ADR-0001 Source of Truth = /docs/hub
- **Date**: 2024-01-15
- **Status**: Accepted
- **Context**: Team required a canonical location for operational documentation.
- **Decision**: Designate `docs/hub/` as the authoritative source of truth for Codex Hub materials.
- **Consequences**: All downstream docs should reference or sync from the hub; changes outside the hub must be mirrored here.
