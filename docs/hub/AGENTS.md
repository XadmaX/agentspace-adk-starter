# AGENTS Instructions for Codex Hub

These guidelines apply to every document inside `docs/hub/` and its subdirectories. If a repository-wide `AGENTS.md` is added later, follow it first and treat these notes as an extension.

## Style
- Prefer concise sections with clear headings and bullet lists.
- Use ISO dates (YYYY-MM-DD) for timelines and decision records.
- Keep internal links relative to `docs/`.

## Maintenance
- Update the roadmap and tasks after planning or sprint reviews.
- Record any decision that affects the hub structure or documentation sources in `DECISIONS.md`.
- When adding prompts or operational guides, ensure they are actionable and versioned in the changelog.
